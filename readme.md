# Quick Search Command(qsc)

`qsc`可以让你在命令提示行的环境中搜索，并将相应的网站使用chrome打开。

### 使用环境

- Windows
- Chrome浏览器

### 安装方法

详情见[安装方法](https://github.com/HarryXD2018/CommandLineSearch/wiki/%E5%AE%89%E8%A3%85)

### 使用样例

~~~command
qsc python -e baidu
~~~
这行命令将会在chrome中使用百度搜索python.
